# ICP备案完整指南

完成ICP备案，让您的域名可以在国内服务器上使用。

## 📋 备案基础知识

### 什么是ICP备案？

ICP备案（Internet Content Provider）是中国工信部对网站的管理制度：
- 使用国内服务器的网站必须备案
- 备案后获得备案号（如：京ICP备12345678号）
- 需要在网站底部显示备案号

### 为什么要备案？

```
国内服务器 + 域名访问 = 必须备案

不备案的后果：
❌ 域名被阻断，无法访问
❌ 服务器可能被暂停
❌ 罚款风险
```

### 备案类型

**个人备案：**
- 适用于：个人网站、博客、私有云
- 限制：不能有商业内容、不能有论坛
- 主体：个人身份证
- 费用：免费（但可能需要邮寄费等）

**企业备案：**
- 适用于：企业网站、商业服务
- 限制：较少
- 主体：营业执照
- 费用：免费

## 📦 第1步：准备材料

### 个人备案所需材料

**必备材料：**

1. **身份证正反面照片**
   - 格式：JPG/PNG
   - 大小：< 4MB
   - 要求：清晰，四角完整，无反光

2. **手持身份证照片**
   - 要求：
     - 手持身份证人像面
     - 身份证信息清晰可见
     - 人脸清晰可见
     - 建议白色背景

3. **网站负责人照片**
   - 要求：
     - 半身照，免冠
     - 纯色背景（白色或蓝色）
     - 穿着得体
     - 可以与手持身份证照片相同

4. **域名证书**
   - 从阿里云域名控制台下载
   - 路径：域名控制台 → 域名列表 → 管理 → 域名证书

5. **服务器信息**
   - 在阿里云自动获取
   - 包括：服务器IP、服务商（阿里云）

**可能需要的材料（部分省份）：**

6. **《网站备案真实性核验单》**
   - 阿里云系统生成
   - 需要打印、签字、扫描上传

7. **暂住证/居住证**（如果不在户籍所在地）
   - 部分省份要求
   - 需要与身份证地址匹配或提供居住证明

### 企业备案所需材料

1. 营业执照照片/扫描件
2. 法人身份证正反面
3. 网站负责人身份证正反面
4. 网站负责人照片
5. 域名证书
6. 服务器信息
7. 授权书（如果负责人不是法人）

## 🖥️ 第2步：阿里云备案系统操作

### 2.1 进入备案系统

```
方法1：
访问：https://beian.aliyun.com/

方法2：
阿里云控制台 → 备案管理

方法3（手机）：
阿里云APP → 备案
```

### 2.2 选择备案类型

点击 **"首次备案"**

如果之前备案过其他网站，选择：
- **"新增网站"**（同一主体）
- **"接入备案"**（更换服务商）

### 2.3 填写主体信息

**个人备案：**

```
主体负责人信息：
├── 姓名：与身份证一致
├── 证件类型：中国居民身份证
├── 证件号码：身份证号
├── 证件住所：身份证地址
├── 通信地址：实际居住地址
├── 手机号码：本人手机号
├── 电子邮箱：常用邮箱
└── 应急手机：备用号码（可选）

注意事项：
⚠️ 手机号必须本人实名
⚠️ 一个手机号只能备案一个主体
⚠️ 通信地址要详细到门牌号
```

### 2.4 填写网站信息

```
网站基本信息：
├── 网站名称：个人网站（见命名规则）
├── 网站内容：个人网站
├── 域名：piowncloud.com
├── 云服务：选择您的轻量应用服务器
└── 网站语言：中文简体

网站负责人信息：
├── 同主体负责人（通常勾选此项）
└── 或单独填写网站负责人

前置审批：
└── 无（个人网站通常不需要）
```

**网站名称命名规则（重要！）**

个人网站命名要求：
```
✅ 可以：
- XX的个人网站
- XX个人博客
- XX的网络日志

❌ 不可以：
- 包含"公司"、"企业"等企业性质词汇
- 包含"论坛"、"社区"等需要专项审批的词汇
- 包含"新闻"、"出版"等需要资质的词汇
- 涉及商业宣传
- 涉及行业、产品名称
```

**推荐命名（针对私有云）：**
```
建议1：李某某的个人网站
建议2：个人网络存储
建议3：XX的个人博客
建议4：XX的网络日志

不要写：
❌ XX云盘（涉及产品）
❌ 私有云存储（涉及行业）
❌ XX网盘（涉及产品）
```

### 2.5 上传材料

按要求上传准备好的材料：
1. 身份证正反面
2. 手持身份证照片
3. 网站负责人照片
4. 域名证书
5. 其他材料（如需要）

**照片要求：**
- 格式：JPG、JPEG、PNG
- 大小：单张 < 4MB
- 分辨率：≥ 600×600像素
- 清晰度：所有文字清晰可辨

### 2.6 真实性核验

**电子化核验（推荐）：**
1. 使用阿里云APP扫码
2. 人脸识别
3. 自动核验通过

**幕布核验（传统方式）：**
1. 申请阿里云幕布（免费邮寄）
2. 收到幕布后拍照
3. 上传幕布照片
4. 等待审核

### 2.7 提交初审

检查所有信息无误后，点击 **"提交备案"**

### 3.1 阿里云审核电话

**时间：** 提交后1-2个工作日内

**来电：** 0571开头（杭州）或95187

**验证内容：**
```
1. 核实身份
   Q: 您是XXX本人吗？
   A: 是的

2. 确认网站信息
   Q: 您要备案的域名是？
   A: piowncloud.com

3. 确认网站用途
   Q: 这个网站是做什么的？
   A: 个人网站，用于个人文件存储和分享
   
   ⚠️ 不要说：网盘、云盘、商业用途

4. 确认网站内容
   Q: 网站有没有论坛、电商等功能？
   A: 没有，只是个人使用

5. 确认主办者信息
   Q: 您的手机号是？通信地址是？
   A: 如实回答（需与申请表一致）
```

**注意事项：**
- ✅ 保持电话畅通
- ✅ 回答与申请表一致
- ✅ 语气自然，不要紧张
- ❌ 不要提商业用途
- ❌ 不要提涉及审批的功能

### 3.2 管局审核电话（部分省份）

**时间：** 提交管局后3-7天内

**来电：** 本省区号

**内容：** 与阿里云电话类似，再次核实信息

## 🏛️ 第4步：管局审核

### 4.1 提交管局

阿里云初审通过后，会自动提交到省通信管理局

**不同省份审核时长：**
```
快的省份：7-10个工作日
  如：浙江、广东、上海

一般省份：10-15个工作日
  如：江苏、北京、福建

慢的省份：15-20个工作日
  如：河北、山东、四川
```

### 4.2 查看审核进度

```
方法1：阿里云备案系统
  https://beian.aliyun.com/
  → 我的备案 → 查看进度

方法2：短信通知
  每个阶段都会收到短信

方法3：邮件通知
  发送到您填写的邮箱
```

### 4.3 审核状态

```
提交中 → 等待几分钟
待初审 → 阿里云审核中（1-2天）
初审通过 → 已提交管局
管局审核中 → 等待管局审核（7-20天）
通过 → 备案完成！
```

## 🎉 第5步：备案完成

### 5.1 获得备案号

审核通过后，您会收到：

**短信通知：**
```
【工信部】恭喜您，您的ICP备案已通过审核。
备案号：京ICP备12345678号-1
```

**邮件通知：**
包含详细的备案信息

### 5.2 查询备案信息

**工信部备案查询：**
```
https://beian.miit.gov.cn/
→ 公共查询
→ 备案信息查询
→ 输入域名或备案号
```

### 5.3 网站添加备案号

**法律要求：**
- 必须在网站首页底部显示备案号
- 备案号必须链接到工信部网站

**具体操作：**

编辑前端文件：

```bash
# 在树莓派上
sudo nano /var/www/raspberrycloud/index.html
```

在页面底部添加：

```html
<footer>
  <p>
    <a href="https://beian.miit.gov.cn/" target="_blank" rel="noopener">
      京ICP备12345678号-1
    </a>
  </p>
</footer>
```

或者在CSS中添加样式：

```html
<footer style="text-align: center; padding: 20px; color: #666;">
  <a href="https://beian.miit.gov.cn/" 
     target="_blank" 
     rel="noopener" 
     style="color: #666; text-decoration: none;">
    京ICP备12345678号-1
  </a>
</footer>
```

## 🔄 第6步：启用域名访问

备案通过后，修改Nginx配置启用域名：

```bash
# 在树莓派上

# 1. 编辑Nginx配置
sudo nano /etc/nginx/sites-available/raspberrycloud

# 2. 修改 server_name
#    从：server_name _;
#    改为：server_name piowncloud.com;

# 3. 测试配置
sudo nginx -t

# 4. 重新加载
sudo systemctl reload nginx
```

**测试访问：**
```
http://piowncloud.com
https://piowncloud.com（如果配置了SSL）
```

## ⚠️ 常见问题和注意事项

### Q1: 备案期间网站能访问吗？

**首次备案：**
- ❌ 域名无法访问（被阻断）
- ✅ 可以用IP访问：http://39.104.94.53

**新增网站/接入备案：**
- ✅ 原有网站仍可访问
- ⏳ 新域名等备案通过后才能访问

### Q2: 备案被驳回怎么办？

**常见驳回原因：**

1. **网站名称不合规**
   ```
   问题：包含商业、行业词汇
   解决：改为"XX的个人网站"
   ```

2. **照片不清晰**
   ```
   问题：身份证信息看不清
   解决：重新拍照上传
   ```

3. **手机号问题**
   ```
   问题：手机号未实名或已备案
   解决：更换手机号或注销原备案
   ```

4. **通信地址不详细**
   ```
   问题：只写到区/县
   解决：详细到门牌号
   ```

**处理方式：**
1. 查看驳回原因（短信/邮件/系统）
2. 修改相应信息
3. 重新提交审核
4. 时间重新计算

### Q3: 备案期间可以更改信息吗？

```
初审前：✅ 可以修改
初审后：❌ 不能修改，需要撤回重新提交
管局审核中：❌ 不能修改，需要等结果
```

### Q4: 备案后需要注意什么？

**必须做的：**
- ✅ 网站底部显示备案号
- ✅ 备案号链接到工信部
- ✅ 网站内容与备案一致

**不能做的：**
- ❌ 网站内容与备案不符
- ❌ 个人备案做商业网站
- ❌ 网站有违法违规内容

**定期维护：**
- 📝 信息变更要及时更新备案
- 📞 保持备案手机号畅通
- 🔍 每年配合管局抽查

### Q5: 备案后可以换服务器吗？

**情况1：同一服务商**
- 阿里云 → 阿里云：不需要操作

**情况2：不同服务商**
- 阿里云 → 腾讯云：需要"接入备案"
- 流程：在新服务商处提交接入备案
- 时间：3-7个工作日

### Q6: 个人备案可以放什么内容？

**可以：**
- ✅ 个人博客
- ✅ 个人相册
- ✅ 个人文件存储（私有云）
- ✅ 个人学习笔记
- ✅ 个人简历展示

**不可以：**
- ❌ 电商交易
- ❌ 企业宣传
- ❌ 论坛社区
- ❌ 新闻资讯
- ❌ 视频直播
- ❌ 金融服务

**您的私有云：**
- ✅ 定位为"个人文件存储"
- ✅ 仅自己和家人使用
- ❌ 不要对外提供服务
- ❌ 不要宣传为"网盘"
