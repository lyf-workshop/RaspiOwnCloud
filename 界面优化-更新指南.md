## 🚀 快速更新（3步）

### 步骤1: 在Windows提交更新



完成！打开浏览器刷新页面（Ctrl+F5 强制刷新），即可看到新界面！

## 🔧 自定义主题

### 修改主色调

编辑 `/var/www/raspberrycloud/css/style.css`：

```css
:root {
    /* 修改这里的颜色 */
    --primary-color: #667eea;  /* 主色 */
    --primary-light: #7c94f5;  /* 浅色 */
    --primary-dark: #5568d3;   /* 深色 */
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

## 🐛 常见问题

### Q: 样式没有更新？
```bash
# 强制清除浏览器缓存
Ctrl+Shift+Delete

# 强制刷新页面
Ctrl+F5

# 检查文件是否更新
ls -lh /var/www/raspberrycloud/css/style.css
```

### Q: 动画卡顿？
可能的原因：
- 浏览器版本过旧
- 设备性能不足
- 开启了过多浏览器扩展

解决方案：
- 更新浏览器到最新版本
- 关闭不必要的扩展

### Q: 想要调整动画速度？
编辑 CSS 中的 `transition-*` 变量：
```css
:root {
    --transition-fast: 150ms ease;    /* 快速 */
    --transition-normal: 250ms ease;  /* 正常 */
    --transition-slow: 350ms ease;    /* 缓慢 */
}
```
